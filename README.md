# Excel Sales Dashboard

## Overview
Interactive Excel dashboard analyzing monthly sales performance across categories, cities, and sales representatives.

## Features
- **Dynamic Slicers**: Filter by category, city, and sales rep
- **Sales Analysis**: Monthly revenue trends and performance metrics
- **Product Performance**: Top-selling products visualization
- **Regional Insights**: Geographic performance distribution
- **Interactive Charts**: Visual representation of KPIs

## Technologies
- Microsoft Excel
- Pivot Tables & Charts
- Data Slicers
- Conditional Formatting

## Files
- Dashboard (Sales data) project.xlsx - Main dashboard file
- README.md - This file

## How to Use
1. Open the Excel file
2. Use slicers to filter data
3. Review charts and tables for insights
